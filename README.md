Game-market---analysis  
Project Overview: 
This project analyzes the structure of the video game market using SQL, Python, and Power BI. 
The goal was to examine market scale, segment distribution, publisher concentration, and pricing differences between platforms. 
The project combines data modeling, analytical SQL queries, and interactive dashboard reporting.  

Analytics:      
Analyze market size (number of games and publishers)     
Measure market concentration (Top 3 segments and publishers)     
Compare PC vs Console pricing     
Identify Year-over-Year trends by genre     
Examine average and median price differences  

Technology:      
SQL,     
Python,     
Power BI     
Data modeling and transformation
